# Posty!
## A demo app that is meant as an pratical introduction to the PHP framework Laravel.



### Features:
1. Registration
2. Login
3. Ability to poste
4. Abiltiy to like/unlike posts
5. View posts per user
6. View all posts
7. View individual posts
8. Delete own posts


### Usage:
